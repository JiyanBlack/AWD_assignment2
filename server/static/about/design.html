Architecture

We have several important architecture and decisions:

1. We decide to user React.js as our main front-end framework together with Webpack.

2. Websocket is responsible for all front-back end communications.

3. MongoDB and Mongoose is our database.

4. Sessions are used to keep a secure communication including real-time chat.

5. Babel and Webpack are responsible for pre-compile all front-end js from ES6 to more compatible ES5 style.
